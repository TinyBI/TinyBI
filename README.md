# TinyBI

[![Travis Build Status](https://travis-ci.org/TinyBI/TinyBI.svg?branch=master)

Openource BI Report System, simple and powerful

### Features
- Scheduled Concurrent Tasks
- RESTFUL APIs
- Multilingual WEB UI

### Business Modules
- General Ledger
